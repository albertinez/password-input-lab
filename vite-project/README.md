# Password Input Lab

## Description
A simple React app demonstrating event handling for:
- Password input typing
- Mouse hover detection on a submit button

## How to Run
1. Clone the repository
2. Install dependencies:
   npm install
3. Start the app:
   npm run dev

## Components
- PasswordInput: Tracks typing events
- SubmitButton: Tracks mouse enter and leave events
